📊 RoseTO Dashboard – Sales & Production Reports

This project contains dashboards and reports developed for Rose Marketing Company to monitor and analyze Sales and Production performance.

The solution was built to provide clear insights into business operations through structured reporting and data visualization.

📁 Repository Structure

The repository contains the following folders:

Sales  – Dashboards and queries related to sales analysis.

Purchase  – Dashboards and queries related to purchase tracking and supplier activity.

🔒 Data Privacy Notice

The files included in this repository are demo versions prepared for portfolio and demonstration purposes.

To protect confidential business information:

Database names have been masked

User Defined Function (UDF) fields have been hidden

Sensitive queries and internal identifiers have been modified

These adjustments ensure that no proprietary company data is exposed while still demonstrating the report structure and logic.

🎯 Purpose of the Project

Track and analyze sales performance

Monitor production and purchase activities

Provide business insights through dashboard reporting

Demonstrate data reporting and query structuring

🛠️ Usage

This repository is intended for:

Demonstration of reporting and dashboard development

Portfolio showcase of business reporting solutions

Learning and reference for report structure and query design
